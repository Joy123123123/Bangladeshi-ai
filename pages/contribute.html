Contribute to Bangladeshi-AI

আমাদের উন্মুক্ত প্রজেক্টে অবদান রেখে বাংলা ভাষার AI শিক্ষার উন্নয়নে সাহায্য করুন।

কীভাবে অবদান রাখবেন?

📝

কন্টেন্ট লিখুন

বাংলায় NLP টিউটোরিয়াল, ব্লগ পোস্ট বা ডকুমেন্টেশন লিখুন।

💻

কোড করুন

নতুন ফিচার, বাগ ফিক্স বা UI উন্নতিতে কোড অবদান রাখুন।

🗄️

ডেটা শেয়ার করুন

বাংলা ডেটাসেট, লেবেলড ডেটা বা টেক্সট কর্পাস শেয়ার করুন।

🐛

বাগ রিপোর্ট করুন

সাইটে কোনো সমস্যা পেলে GitHub Issues-এ রিপোর্ট করুন।

🎨

ডিজাইন করুন

UI/UX উন্নতি, আইকন বা ভিজ্যুয়াল কন্টেন্ট তৈরি করুন।

🌍

ছড়িয়ে দিন

প্রজেক্টটি সোশ্যাল মিডিয়া ও সম্প্রদায়ে শেয়ার করুন।

GitHub-এ অবদানের ধাপ

১

Repository Fork করুন

GitHub-এ গিয়ে "Fork" বাটনে ক্লিক করুন।

Fork করুন
২

Clone করুন

আপনার fork করা repository লোকালি clone করুন।

git clone https://github.com/YOUR_USERNAME/Bangladeshi-ai.git cd Bangladeshi-ai
৩

Feature Branch তৈরি করুন

আপনার পরিবর্তনের জন্য একটি নতুন branch তৈরি করুন।

git checkout -b feature/my-bengali-feature
৪

পরিবর্তন করুন ও Commit করুন

আপনার পরিবর্তন করুন এবং commit করুন।

git add . git commit -m "Add: Bengali NLP tutorial for beginners"
৫

Pull Request খুলুন

GitHub-এ গিয়ে Pull Request খুলুন এবং বর্ণনা দিন।

PR খুলুন

অবদান নির্দেশিকা

করুন

  • বাংলা ভাষায় কন্টেন্ট লিখুন
  • পরিষ্কার ও বোধগম্য কোড লিখুন
  • তথ্যসূত্র ও রেফারেন্স উল্লেখ করুন
  • সৌহার্দ্যপূর্ণ আলোচনায় অংশ নিন

করবেন না

  • কপিরাইটযুক্ত কন্টেন্ট যোগ করবেন না
  • ব্যক্তিগত তথ্য শেয়ার করবেন না
  • অসম্মানজনক বা আপত্তিজনক কন্টেন্ট
  • অপরীক্ষিত বা ভুল তথ্য যোগ করবেন না
