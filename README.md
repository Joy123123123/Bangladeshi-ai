## 📖 Project Overview

**Bangladeshi-AI** is a comprehensive web platform that bridges Bengali language processing and artificial intelligence education. Our mission is to make Bengali NLP accessible to students, educators, and researchers across Bangladesh and the global Bengali-speaking community.

### 🎯 Vision
To create the most comprehensive Bengali-language AI education hub, empowering learners with modern tools, curated datasets, and cutting-edge research resources.

---

## ✨ Features

### Current Features
- 🏠 **Professional Homepage** – Overview of Bengali AI tools and resources
- 📝 **Text Analyzer** – Bengali text sentiment analysis and linguistic insights
- 🎙️ **Speech-to-Text** – Bengali voice recognition demonstration
- 🔄 **Bengali Translator** – Bengali ↔ English translation with explanations
- 📚 **Datasets Hub** – Curated links to Bengali NLP datasets
- 🎓 **Tutorials** – Step-by-step learning resources for Bengali NLP
- 🔬 **Research Papers** – Academic references and Bengali AI publications

### Planned Features
- 🤖 AI-powered Bengali grammar checking
- 📊 Named Entity Recognition (NER) for Bengali
- 🎯 Interactive Bengali learning exercises
- 💬 Bengali chatbot integration
- 📱 Progressive Web App (PWA) support
- 🌐 API endpoints for Bengali NLP tasks

## 🚀 Getting Started

### Option 1: View Online
Visit the live site at: [https://joy123123123.github.io/Bangladeshi-ai/](https://joy123123123.github.io/Bangladeshi-ai/)

### Option 2: Run Locally
```bash
# Clone the repository
git clone https://github.com/Joy123123123/Bangladeshi-ai.git

# Navigate into the directory
cd Bangladeshi-ai

# Open in your browser
open index.html
# or on Linux:
xdg-open index.html
# or use a local server:
python3 -m http.server 8000
```

Then visit `http://localhost:8000` in your browser.

## 📚 Bengali NLP Resources

### Datasets
- [Bengali.AI](https://bengali.ai) – Open Bengali language datasets
- [CC-100 Bengali](https://data.statmt.org/cc-100/) – Massive Bengali web corpus
- [IndicNLP](https://indicnlp.ai4bharat.org/) – Indic languages NLP resources
- [Kaggle Bengali Datasets](https://www.kaggle.com/datasets?search=bengali) – Community datasets

### Tools & Libraries
- [bnlp-toolkit](https://github.com/sagorbrur/bnlp) – Bengali NLP toolkit
- [BanglaBERT](https://huggingface.co/csebuetnlp/banglabert) – Bengali BERT model
- [Bangla2Vec](https://github.com/sagorbrur/bangla2vec) – Bengali word embeddings

### Research
- [arXiv Bengali NLP Papers](https://arxiv.org/search/?query=bengali+nlp&searchtype=all)
- [ACL Anthology](https://aclanthology.org/) – NLP/Computational linguistics papers
